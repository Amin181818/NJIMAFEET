3. Amin — Détection de chute
Fichier conseillé

fall_detection.c

Tâche

Analyser :

les données du corps
les données du terrain

puis décider du niveau de danger.

Ce qu’il doit coder
logique de décision
seuils
états possibles :
normal
warning
risque
chute imminente
Structure possible
typedef enum {
    STATE_NORMAL,
    STATE_WARNING,
    STATE_FALL_RISK,
    STATE_FALL_IMMINENT
} FallState;
Exemples de règles
forte inclinaison + faible pression stable → warning
obstacle + accélération anormale → fall risk
trou + déséquilibre important → fall imminent
Objectif

Être le “cerveau sécurité” du système.

Priorité

60

4. Jolyon — Stabilisation et alertes

Ce qu’il doit coder

Structure possible
typedef struct {
    int stabilization_on;
    int ankle_lock_on;

    int buzzer_on;

Logique

STATE_FALL_RISK → vibration forte + stabilisation

Fichiers à prévoir

Je vous conseille cette structure :

main.c
shared_data.h
sensor_simulation.c
mapping.c
fall_detection.c
stabilization_alert.c
display_ui.c
Rôle de main.c

Le main :

initialise les données
crée les threads
lance chaque tâche
gère éventuellement les mutex
attend les threads si besoin
Rôle de shared_data.h

Dans ce fichier, vous mettez :

toutes les structures partagées
les variables globales
les prototypes des fonctions
les mutex si vous en utilisez

Exemple :

#ifndef SHARED_DATA_H
#define SHARED_DATA_H

#include <pthread.h>

typedef struct {
    float accel;
    float tilt;
    float gyro;
    float pressure_left;
    float pressure_right;
    float depth;
    int mode_scenario;
} SensorData;

typedef struct {
    int hole_detected;
    int obstacle_detected;
    int slippery_surface;
    float terrain_slope;
    float obstacle_distance;
} MapData;

typedef enum {
    STATE_NORMAL,
    STATE_WARNING,
    STATE_FALL_RISK,
    STATE_FALL_IMMINENT
} FallState;

typedef struct {
    int stabilization_on;
    int ankle_lock_on;
    int vibration_level;
    int buzzer_on;
    int led_state;
} ActuatorState;

extern SensorData sensor_data;
extern MapData map_data;
extern FallState fall_state;
extern ActuatorState actuator_state;

extern pthread_mutex_t data_mutex;

#endif
